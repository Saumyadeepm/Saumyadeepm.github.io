GUI Browser using python

Description

Created a GUI based Browser by using PyQT library in Python Programming Language. Embedded with all the features of what an modern browser should have, like forward, backward, stop, go, URL bar, home, new-tab, switch-tabs, double-click to open new-tab, SSL checker and personalized dynamic theme changing feature.

It was an end semester assignment project, in which we were supposed to create any project in python by utilizing whatever we have learnt in the semester.
